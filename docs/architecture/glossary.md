# Glossary

- **Artifact**: A generated document output (PRD, ADR, SPEC, roadmap, prompts).
- **Run**: A durable multi-step pipeline execution for a project.
- **Step**: One discrete stage in a run (research, competitor scan, PRD draft, etc.).
- **Citation**: A record linking a claim to a source URL and excerpt.
- **Chunk**: A retrieval unit derived from uploaded content or generated artifacts.
- **Agent mode**: A configured agent with a specific role and tool allowlist.
- **Tool**: A callable capability provided to an agent (web search, retrieval, sandbox).
- **Dynamic tool**: A tool injected only when needed to reduce context bloat.
